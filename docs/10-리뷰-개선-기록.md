## 2. 이번 개선 작업

### 2.1 FuzzyWuzzy → Apache Commons Text 교체

**문제:** `me.xdrop:fuzzywuzzy:1.4.0`은 2021년 이후 유지보수 중단된 라이브러리.

**변경 내용:**

| 항목 | 변경 전 | 변경 후 |
|------|---------|---------|
| 의존성 | `me.xdrop:fuzzywuzzy:1.4.0` | `org.apache.commons:commons-text:1.13.0` |
| 유사도 계산 (base) | `FuzzySearch.tokenSetRatio()` | `LevenshteinDistance` 기반 정규화 유사도 |
| 유사도 계산 (extra) | `FuzzySearch.weightedRatio()` | `JaroWinklerSimilarity` |

**수정 파일:**
- `build.gradle` — 의존성 교체
- `src/main/java/com/apiagent/recipe/RecipeStore.java` — import 및 similarity() 메서드

**검증:** 64개 테스트 전부 통과. 유사 레시피 제안 테스트(`유사_레시피_제안`)에서 동일한 정렬 결과 확인.

### 2.2 문서 최신화

**수정 파일 및 내용:**

`README.md`:
- 기술 스택 버전: Java 21→25, Spring Boot 3.4.4→4.0.3, Spring AI 1.0.3→2.0.0-M3
- FuzzyWuzzy→Apache Commons Text, Gradle 8.12→8.14.1
- `java --enable-preview -jar ...` → `java -jar ...`

- 테스트 수: 11개 파일/52개 → 15개 파일/64개
- context 패키지 설명: ThreadLocal holder → ScopedValue holder

`docs/02-설치-및-실행.md`:
- 주요 의존성 표 버전 업데이트

- Docker 베이스 이미지: temurin:21 → temurin:25

## 3. Spring AI 2.0.0-M3 호환성 점검 결과

### 3.1 안전 확인된 항목

| 점검 항목 | 결과 | 근거 |
|-----------|------|------|
| MCP 어노테이션 패키지명 | ✅ 안전 | `org.springframework.ai.tool.annotation.Tool` 등 2.0 표준 패키지 사용 |
| ToolContext 대화 히스토리 제거 | ✅ 영향 없음 | ToolContext 미사용. `List<Message>`로 직접 관리 |
| OpenAI 기본 모델 변경 (gpt-5-mini) | ✅ 안전 | `application.yml`에서 `gpt-4o` 명시적 설정 |

| ToolResponseMessage | ✅ 안전 | `.builder().responses().metadata().build()` 2.0 빌더 패턴 사용 |

### 3.2 향후 검토 필요 항목

**에이전트 내부 도구 관리 방식 (낮은 위험)**

현재 AgentRunner는 도구를 `Map<String, Function>` 콜백으로 수동 관리합니다.
Spring AI는 `FunctionToolCallback` 래퍼를 통한 프레임워크 관리 방식을 권장합니다.

```java
// 현재 방식 (AgentRunner.java)
Map<String, Function<Map<String, Object>, String>> toolCallbacks = ...;
var callback = toolCallbacks.get(toolName);
var result = callback.apply(args);

// Spring AI 권장 방식 (향후 고려)
FunctionToolCallback.builder("graphql_query", 실행함수)
    .description("GraphQL 쿼리를 실행합니다")
    .inputType(Map.class)
    .build();
```

**현재 판단:** M3(마일스톤) 단계에서 성급한 전환은 GA에서 재변경 위험이 있으므로, **Spring AI GA 릴리스(예상 2026 Q2-Q3) 이후 재검토** 권장.
